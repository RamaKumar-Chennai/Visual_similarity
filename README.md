## 🧠 Visual Product Similarity & Image-Based Recommendation System (Amazon-Style)

This project builds a **computer vision–based recommendation engine** that identifies and suggests visually similar products using deep learning embeddings. It mimics Amazon’s “Similar Items” feature, enabling image-based product search and improving discovery when text metadata is noisy or incomplete.

### 🎯 Objectives
- Extract high-quality visual embeddings using **ResNet18**.  
- Build a fast **FAISS similarity search engine** for millisecond retrieval.  
- Rank and display **Top‑K visually similar products** for any query image.  
- Evaluate performance using **visual inspection**.

### 🛠 Tech Stack
- **Python**, **PyTorch**, **FAISS** 
- **Deep Learning** for feature extraction  
- **Vector Databases** for similarity search  

### 📊 Dataset
-FASHIONMNIST dataset

### 📈 Results

- Sub‑second search latency using FAISS  
- Visually coherent recommendations across fashion and lifestyle categories
